React on SVG Ngraph (webgl) Cytoscape
Description The most simple approach data is parsed and added to DOM by react. High performance 3d graph library. Comprehensive library based on Canvas.
Advantages Does not crash for big models.
  • The performance is good even for really big models.
  • No longer maintained.
  • Implements folding algorithm (for predefined groups).
  • Well maintained, in active development.
Disadvantages DOM structure is slow and there is noticeable drop in performance with increasing numbers of elements.
  • Questionable usability of 3d graphs.
  • More challenging to render additional elements - render in webgl.
  • Performance problems - tends to lag on test model.
